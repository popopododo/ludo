# LUDO: Game Console Auction Platform

## 초기 라이브러리 설정

1. 설치 프로그램을 다운로드 받아 실행 https://getcomposer.org/Composer-Setup.exe
2. 프로젝트 ROOT에서 아래 명령어 실행

```bash
composer install
``` 

## scss 사용법

1. ``staitc/scss`` 경로에 ``scss_cache`` 디렉토리 생성
2. ``static/scss/app.scss``에 scss 작성
